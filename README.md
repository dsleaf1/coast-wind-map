# Coast Wind Map

HRDPS 2.5 km wind for the BC/WA coast — Puget Sound to Dixon Entrance, including
all of Haida Gwaii. One continuous map, ~48 h timeline, 4 model runs/day.

Live: https://dsleaf1.github.io/coast-wind-map/

Wind only. For currents and wind-vs-current roughness in the Salish Sea see the
[Unified Marine Map](https://dsleaf1.github.io/salish-marine-map/).

Installable as a phone PWA with offline forecast save (one whole-coast wind pack).

## Deploy

Source of truth lives in `cape_st_james/`; this repo is a deploy target:

    cp cape_st_james/coast_wind_map.html        coast-wind-map/index.html
    cp cape_st_james/coast_manifest.webmanifest coast-wind-map/
    cp cape_st_james/coast_sw.js                coast-wind-map/
    git -C coast-wind-map commit -am "…" && git -C coast-wind-map push

Bump `CACHE_VERSION` in `coast_sw.js` whenever the page or a shell asset changes,
or returning users keep the cached shell. Icons are generated by
`cape_st_james/coast_make_icons.py` and are tracked here only (cape's .gitignore
excludes *.png).

Data comes from the [hrdps-ingest](https://github.com/dsleaf1/hrdps-ingest)
pipeline's `coast-overview` region, served from R2.
